--- Slide 9 ---
Basic operations
df['NewCol'] = df['Age'] + 5           # add new column
df.drop('NewCol', axis=1, inplace=True)  # drop column
df.rename(columns={'Name': 'FullName'}, inplace=True)  # rename
df.sort_values('Age', ascending=False)  # sort

--- Slide 10 ---
Data Cleaning Steps--1. Handle Missing Values
df.isnull().sum()               # count nulls
df.dropna(inplace=True)         # drop rows with any null
df.fillna(0, inplace=True)      # fill null with 0
df['Age'].fillna(df['Age'].mean(), inplace=True)  # fill with mean

--- Slide 11 ---
Data Cleaning Steps--2.Handle Duplicates
df.duplicated().sum()      # how many duplicates
df.drop_duplicates(inplace=True)   # remove duplicates

--- Slide 12 ---
Data Cleaning Steps--3.Handle Data Types
df.dtypes                  # check types
df['Age'] = df['Age'].astype(int)   # convert type
df['Date'] = pd.to_datetime(df['Date'])  # convert to datetime

--- Slide 13 ---
Data Cleaning Steps--4. Handle Outliers
Detect:
df['Age'].describe()     # check min, max, quartiles
Filter out:
df = df[df['Age'] < 100] # remove unreasonable ages

--- Slide 14 ---
Data Cleaning Steps—5. String Operations
df['City'] = df['City'].str.strip()       # remove whitespace
df['City'] = df['City'].str.lower()       # to lowercase
df['City'] = df['City'].str.replace('mumbai','Mumbai')

--- Slide 15 ---
Data Cleaning Steps—6. Encoding Categorical Variables
# One-hot encoding

# Label encoding
from sklearn.preprocessing import LabelEncoder

df['City_encoded'] = le.fit_transform(df['City'])

Matplotlib & Seaborn
Data Visualization in Python

--- Slide 2 ---
Data Visualization in Exploratory Data Analysis (EDA)
EDA is the first step in analyzing any dataset.
It helps you understand the structure, relationships, and patterns in the data.
It includes: Descriptive statistics (mean, median, std. deviation, etc.)Data visualization (graphs, plots, charts)

--- Slide 3 ---
Role of Data Visualization in EDA
Visualization makes it easy to:
Detect patterns and trends
Spot outliers/anomalies
Understand distributions
Compare variables (correlations & relationships)

--- Slide 4 ---
Common Visualization Techniques for EDA
1. Univariate Analysis (Single Variable)
Histogram → Distribution of a numeric variable
Boxplot → Spread, quartiles, outliers
Bar chart → Frequency of categorical data

2. Bivariate Analysis (Two Variables)
Scatter Plot → Relationship between two numerical variables
Line Plot → Trends over time
Grouped Barplot → Compare categories

3. Multivariate Analysis (Multiple Variables)
Pairplot (Seaborn) → Relationship between multiple variables
Heatmap → Correlation matrix
Stacked Barplot / Multiple Line Plots → Compare across groups

--- Slide 5 ---
Key Insights You Can Get
Distribution → Is the data normal, skewed, or has outliers?
Relationships → Are two variables correlated? (e.g., bill vs tip)
Categorical Comparisons → Which group spends more?
Trends → How values change with time or category.

Graphical User Interfaces (GUIs) in Python allow users to interact with applications through visual elements like buttons, menus, windows, and text boxes, rather than solely through command-line interfaces.
Popular Python GUI Frameworks: 
Tkinter: 
Python's standard and built-in GUI library. 
Provides a lightweight wrapper around the Tcl/Tk toolkit. 
Cross-platform, ensuring applications function similarly across Windows, macOS, and Linux. 
Suitable for developing simple to moderately complex desktop applications and learning GUI programming basics.

--- Slide 3 ---
PyQt: 
A set of Python bindings for the Qt application framework. 
Offers a comprehensive suite of widgets and features for building sophisticated, high-performance applications. 
Supports cross-platform development, including desktop and mobile environments.
wxPython: 
A wrapper for the wxWidgets C++ library. 
Provides a native look and feel across different operating systems. 
Known for its extensive set of widgets and robust features for desktop application development.

--- Slide 4 ---
Kivy: 
Designed for creating modern and responsive user interfaces, particularly for multi-touch applications and mobile devices. 
Employs a declarative language (KV language) for UI design, enhancing development speed.

Widgets: 
The fundamental building blocks of a GUI, such as buttons, labels, entry fields, and scrollbars. 
Layout Management: 
Techniques for arranging widgets within a window, including packing, placing, and gridding. 
Event Handling: 
Mechanisms for responding to user interactions, such as button clicks, key presses, or mouse movements, by binding them to specific functions.

--- Slide 2 ---
Streamlit
Streamlit is an open-source Python library for building interactive web apps using only Python. 
It's ideal for creating dashboards, data-driven web apps, reporting tools and interactive user interfaces without needing HTML, CSS or JavaScript.

--- Slide 3 ---
Streamlit Installation
Make sure Python and pip are already installed on your system. To install Streamlit, run the following command in the command prompt or terminal:


pip install streamlit

--- Slide 4 ---
How to run Streamlit file?
To run a Streamlit app, open the command prompt or Anaconda prompt and type:
streamlit run filename.py
For example, if file name is sample.py, run:
streamlit run sample.py
After running the command, Streamlit starts a local development server and displays a URL (http://localhost:8501). Open this URL in your browser to view and interact with the app.

import streamlit as st
# Title of the app
st.title("BMI Calculator")
# Input: Weight in kilograms
weight = st.number_input("Enter your weight (kg):", min_value=0.0, format="%.2f")
# Input: Height format selection
height_unit = st.radio("Select your height unit:", ['Centimeters', 'Meters', 'Feet'])
# Input: Height value based on selected unit
height = st.number_input(f"Enter your height ({height_unit.lower()}):", min_value=0.0, format="%.2f")
# Calculate BMI when button is pressed
if st.button("Calculate BMI"):
    try:
        # Convert height to meters based on selected unit
        if height_unit == 'Centimeters':
            height_m = height / 100
        elif height_unit == 'Feet':
            height_m = height / 3.28
        else:
            height_m = height

--- Slide 20 ---
# Prevent division by zero
        if height_m <= 0:
            st.error("Height must be greater than zero.")
        else:
            bmi = weight / (height_m ** 2)
            st.success(f"Your BMI is {bmi:.2f}")
# BMI interpretation
            if bmi < 16:
                st.error("You are Extremely Underweight")
            elif 16 <= bmi < 18.5:
                st.warning("You are Underweight")
            elif 18.5 <= bmi < 25:
                st.success("You are Healthy")
            elif 25 <= bmi < 30:
                st.warning("You are Overweight")
            else:
                st.error("You are Extremely Overweight")
    except:
        st.error("Please enter valid numeric values.")
